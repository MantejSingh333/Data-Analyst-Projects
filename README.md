# Data Analysis Using Python in Microsoft Fabric

Project using Python within Microsoft Fabric to clean, analyze, and process data as part of a Microsoft analytics workflow.

Data Analysis Using Python in Microsoft Fabric
Project Overview

This project was completed as part of my academic studies and focuses on using Python within Microsoft Fabric to perform data cleaning, analysis, and transformation. The goal was to gain hands-on experience working in a modern, cloud-based analytics environment aligned with real-world enterprise data workflows.

The project demonstrates how Python can be used inside Microsoft Fabric to explore and analyze datasets efficiently within the Microsoft analytics ecosystem.

Objectives

Work within Microsoft Fabric using Python notebooks

Load, clean, and transform datasets

Perform exploratory data analysis (EDA)

Apply Python-based logic within an enterprise analytics platform

Key Tasks Performed
